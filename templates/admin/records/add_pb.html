How to Add Personal Bests

1. Athlete SDMS: Enter the athlete's SDMS number. The athlete must already exist in the system.

2. Event & Class: Select the event and classification for this performance.

3. Performance: Enter the performance value (time for track events, distance for field events).

4. Location: Where this performance was achieved (city, country).

5. Date: When this performance was achieved.

6. Competition Flag: Check if this was achieved in this competition (usually unchecked for manual entries).
